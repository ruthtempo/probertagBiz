## Web Scraping Challenge

- Getting specific data from a website (data crawling)
- Saving it into supabase (database)
- Fetching data from supabase and displaying it interactively (sorting, calculations, etc.)

## Techstack

 - Puppeteer
 - Supabase
 - NodeJS
 - React 
 - React-Bootstrap
